# v8.5 扩展模式：Unity IL2CPP + Python打包 + VMP分析

## 扩展流程

当用户要求"增加破解技术"时，按以下流程执行：

1. **GitHub预研** — 搜索相关领域开源项目（注意min_stars降级）
2. **技术选型** — 基于预研结果选择技术栈
3. **工具开发** — 创建Python封装工具
4. **参考文档** — 创建对应references/文档
5. **版本更新** — 更新SKILL.md版本历史
6. **Git提交** — 提交并推送到Gitee
